# 📚 Belajarin Aja Yuk

Belajarin Aja Yuk is a modern e-learning mobile application built with Flutter. The platform helps students access learning materials, watch educational videos, complete quizzes, and interact with an AI-powered learning assistant to enhance their learning experience.

---

## 🚀 Features

### Authentication
- User Registration
- User Login
- Secure Authentication

### Learning Module
- Browse Learning Materials
- Video-Based Learning
- Organized Course Categories
- Progress Tracking

### Quiz & Assessment
- Interactive Quizzes
- Score Calculation
- Learning Evaluation

### AI Learning Assistant
- AI-powered chatbot
- Learning guidance and support
- Question and answer assistance

### User Profile
- Profile Management
- Learning Progress Overview
- Account Settings

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/belajarin_aja_yuk.git
```

### Navigate to Project Folder

```bash
cd belajarin_aja_yuk
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
